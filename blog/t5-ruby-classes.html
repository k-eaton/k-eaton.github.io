Classy Cats

  • January 6th, 2015
  • technical, classes, objects, cats

The best comparison I can think of is to a cat. (I know, cliche, but work with me here.) Objects are created by calling an instance of a class. So a class is simply a template for an object. Once defined it can be called over and over again to create new objects. And thus my career as a crazy cat lady is born.

A class typically starts with a method called initialize, which starts the party off by defining some basic variables. These variables will be accessible throughout the object. For example: 

			
class Cat
  def initialize (eyes, color, pattern, name)
   @eyes = eyes
   @color = color
   @pattern = pattern
   @name = name
  end
end

To create a new cat, we need to specify some details. 

			
trouble = Cat.new("green", "orange", "tabby", "Trouble")

And congratulations, we have a new cat! Of course, cats do things. When you need to make an object do something, you create a method in the class. There are two kinds of methods, public and private. A public method is something you can call directly. For example, we can have our cat come when we call it. (Oh quit laughing. It could happen.) So we create the "come here" method. 

			
  def come_here(cat_call)
    if cat_call == "Here kitty kitty!"
      return cat
    end
  end

This is a public method because you can call it. 

			
trouble = Cat.new("green", "orange", "tabby", "Trouble")
trouble.come_here("Here kitty kitty!")

But sometimes there are methods you just canâ€™t access directly. Example: 

			
  private

  def purr
    return purr
  end

The cat can do that, but we can't control it directly. But there is something we can control: 

			
  def pet
    cat.pet
    purr
  end

(You'd have to place this above the word private in the class.)

Now you can pet your cat, and in turn it returns a purr, accessing the method you can't directly control. 

			
trouble = Cat.new("green", "orange", "tabby", "Trouble")
trouble.come_here("Here kitty kitty!")
trouble.pet

So now you too have access to your own crazy cat lady starter kit. 

			
class Cat

  def initialize (eyes, color, pattern, name)
   @eyes = eyes
   @color = color
   @pattern = pattern
   @name = name
  end

  def come_here(cat_call)
    if cat_call == "Here kitty kitty!"
      return cat
    end
  end

  def pet
    cat.pet
    purr
  end

  private

  def purr
    return purr
  end

end

And now you can enjoy your new cat.
